# How to run the project
1. Create a project folder in local directory.
2. Download the project from github repository into the local folder.
3. Open the project in VS code.
4. Open the command prompt and change the directory to the project folder 
   or you can directly open the command prompt into the project folder by 
   clicking Ctrl+Alt+C in the VS code.
5. Run the following commands in the command prompt. 
   i.  npm install
   ii. npm start.
   
This will start the project on localhost:3000 port.
